# Remote IT Support Lab

## Overview
This project demonstrates hands-on remote IT support skills by simulating real-world troubleshooting scenarios using AnyDesk.

## Scenarios Covered
- Slow system performance troubleshooting
- DNS-related internet connectivity issues
- Application crash and recovery

## Tools Used
- AnyDesk
- Task Manager
- Command Prompt

## Skills Demonstrated
- Remote troubleshooting
- Performance optimization
- Network diagnostics
- End-user support

## Project Structure
Each scenario includes:
- Problem description
- Diagnosis steps
- Root cause
- Resolution
- Result

## Outcome
Developed practical experience handling real-world IT support issues in a remote environment.
