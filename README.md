# Remote IT Support Lab

## Overview
This project demonstrates remote IT support skills by simulating real-world troubleshooting scenarios using AnyDesk.

## Key Skills
- Remote troubleshooting
- Performance optimization
- Network issue resolution
- End-user support

## Tools Used
- AnyDesk
- Task Manager
- Command Prompt

## Outcome
Developed practical experience in resolving IT issues in a remote support environment.
